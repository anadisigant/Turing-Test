Como você explicaria o que é um algoritmo para alguém que nunca usou computador?

Eu diria que é tipo uma receita de bolo. Você segue os passos certinhos pra chegar num resultado. No caso do computador, quem escreve essa receita é a gente, e ele só segue. Se errar um passo, a coisa toda dá errado.

Acho que colocaria como se fosse letrinhas escritas de uma maneira muito especifica para mandar o computador resolver um problema muito específico.

Eu diria que um algoritmo é uma série de instruções escrita por um programador e "lida" pela máquina, a fim de realizar determinada tarefa. Pode não estar 100% correto mas a pessoa iria pegar um pouco da ideia.
